# Loon_plugin

Loon 每日智能推送插件：  
1) **AI 技术早报**：偏框架/推理/Serving/LLM 版本迭代（GitHub Releases Atom + 主流框架博客 RSS）  
2) **股票/指数/基金行情**：以「**当日开盘后涨跌**」为主，支持 A 股/ETF/指数/港股/美股 + 场外基金估值
